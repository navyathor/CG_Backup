What is an enum?

An enum defines a user-defined integral type consisting of named integer constants.
 
ðŸ”¹ Syntax:

c

Copy

Edit

enum Color {

    RED,

    GREEN,

    BLUE

};

By default:
 
RED = 0
 
GREEN = 1
 
BLUE = 2
 
You can override values:
 
c

Copy

Edit

enum Status {

    SUCCESS = 0,

    FAILURE = -1,

    TIMEOUT = 100

};

You can also use typedef:
 
c

Copy

Edit

typedef enum {

    OFF,

    ON

} PowerState;

ðŸ”¹ Usage Example:

c

Copy

Edit

PowerState ps = ON;

if (ps == ON) {

    // do something

}

ðŸ”¹ Behind the Scenes:

Enums are stored as int (typically 4 bytes).
 
You can assign any integer value, even values not listed in the enum (not type-safe).
 
ðŸ”¹ Why use enum?

Readability: Meaningful names over magic numbers.
 
Maintainability: Easier to extend/modify states.
 
Debugging: Better debug symbols than raw integers.
 
ðŸ”¹ Advanced: Enum as Bitmask (less common but possible)

c

Copy

Edit

enum Flags {

    FLAG_READ = 1 << 0,

    FLAG_WRITE = 1 << 1,

    FLAG_EXECUTE = 1 << 2

};

int perms = FLAG_READ | FLAG_WRITE;

âœ… What is a union?

A union is a special data type where all members share the same memory location.
 
ðŸ”¹ Syntax:

c

Copy

Edit

union Data {

    int i;

    float f;

    char str[20];

};

All fields share memory, so sizeof(union) = size of the largest member.
 
ðŸ”¹ Example:

c

Copy

Edit

union Data data;

data.i = 10;

printf("%d\n", data.i);
 
data.f = 220.5;

printf("%f\n", data.f);  // now data.i is likely garbage

ðŸ”¹ Use Cases:

Memory efficiency (e.g., embedded systems).
 
Type-punning (accessing the same memory as different types).
 
Variant types / tagged unions (often with an enum to track type).
 
ðŸ”¹ Example: Tagged Union

c

Copy

Edit

typedef enum {

    TYPE_INT,

    TYPE_FLOAT,

    TYPE_STRING

} VarType;
 
typedef struct {

    VarType type;

    union {

        int i;

        float f;

        char str[20];

    } data;

} Variant;
 
void print_variant(Variant v) {

    switch (v.type) {

        case TYPE_INT:   printf("Int: %d\n", v.data.i); break;

        case TYPE_FLOAT: printf("Float: %f\n", v.data.f); break;

        case TYPE_STRING:printf("String: %s\n", v.data.str); break;

    }

}

ðŸ” Differences Between struct and union
 
Feature	struct	union

Memory	Sum of all members	Max of all members

Access	All members can be used	Only one member valid at a time

Use Case	Group related data	Save memory, variant types

ðŸ§  Best Practices (3 Years+ Experience Level)

Use enum for state machines, error codes, protocol states, etc.
 
Use union in protocol stack implementations, hardware registers, or when interfacing with raw data (bit-level).
 
Use typedef enum and typedef union for cleaner code.
 
Avoid writing to one union member and reading from another unless you know the architecture-specific behavior (type-punning is undefined behavior in strict C).
 
ðŸ”¬ Interview-level Questions

Can enums have duplicate values?
 
Yes, but the symbolic names must be unique.
 
c

Copy

Edit

enum {

    A = 1,

    B = 2,

    C = 2 // valid

};

Can we determine which member of a union was last written?
 
Not automatically. You must track it yourself using an enum or flag.
 
Is reading from a union member different from the one written to undefined behavior?
 
Yes, in standard C. Some compilers allow it (like GCC with -fno-strict-aliasing), but it's not portable.
 
Size of union with int and char[10]?
 
sizeof(union { int i; char str[10]; }) = 10 (not 14), due to shared memory.
