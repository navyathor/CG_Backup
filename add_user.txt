  You can make the changes using the following steps:
Go to project: petalinux-config -c rootfs > PetaLinux Rootfs Settings > Add extra users
Provide the users. To add extra users to the PetaLinux system, provide the user ID (userid) and password (passwd) separated by :; for multiple users, separate sets of user IDs and passwords using ;.
Examples:

To add a passwd1 for user1:

user1:passwd1; or user1:passwd1
To add an empty passwd for the user1:

user1:
To add user1 and user2 with passwd1 and passwd2, respectively:

user1:passwd1;user2:passwd2;
